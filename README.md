基于Java的记事本
===========

![预览图](https://raw.githubusercontent.com/muyangren907/Java-NotePad/master/screenshots/1.png)

## 说明

- Java编写的记事本，仅供参考

## 使用

###	①

- 下载源代码

- 编译

```
javac MyNotePad.java -d ./ -encoding utf-8
```

-	运行

```
java MyNotePad
```

### ②

- 下载jar包运行
